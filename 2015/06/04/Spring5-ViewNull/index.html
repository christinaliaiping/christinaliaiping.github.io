摘要：
视图层显示null

时间

2015年3月30日

问题

public ModelAndView onSumit(HttpServletRequest req, Object command) throws Exception {
        HelloWorldTwo helloWorld = (HelloWorldTwo) command;
        Map model = new HashMap();
        model.put("helloWorld", helloWorld.getHelloWorld());
        return new ModelAndView(getViewPage(), model);
    }

如上所示代码，即使model设置了元素“helloWorld”的值，但无法传参数至视图层，视图显示null。

原因

原因是helloWorld的属性helloWorld没有被初始化，仍然是null值。换言之，没有从前台中传递参数至后台。

解决

在model.put(“helloWorld”, helloWorld.getHelloWorld())后，添加语句：
helloWorld.setHelloWorld(req.getParameter("helloWorld"));
