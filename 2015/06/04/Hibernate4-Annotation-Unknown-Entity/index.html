摘要：
根据教程实现annotation的例子，在运行程序时，报错提示Unknown entity

时间

2015年5月29日

问题

根据教程实现annotation的例子，在运行程序时，报错提示Unknown entity，报错内容如下：

原因

根据搜索结果，已排除导入包有误的错误，发现是main函数
factory = new AnnotationConfiguration().configure().addAnnotatedClass(Employee2.class).buildSessionFactory();
写成
factory = new AnnotationConfiguration().configure().addAnnotatedClass(Employee.class).buildSessionFactory();
导致无法映射Employee2，引起提示未知实体错误。

解决

改为factory = new AnnotationConfiguration().configure().addAnnotatedClass(Employee2.class).buildSessionFactory();
即可
